### 🔧 打包技术细节

#### 使用的工具
- **PyInstaller 6.14.1**：Python程序打包工具
- **--onefile**：打包成单个exe文件
- **--windowed**：Windows下隐藏控制台窗口

#### 打包参数
```bash
pyinstaller --onefile --windowed --name=BadgePatternTool \
    --add-data=src;src \
    --hidden-import=PIL \
    --hidden-import=reportlab \
    --hidden-import=PySide6 \
    src/main.py
```

#### 包含的依赖
- **PySide6**：GUI框架
- **Pillow (PIL)**：图像处理
- **reportlab**：PDF生成
- **numpy**：数学计算支持
