### 🔄 如何更换图标

#### 方法1：使用现有图片
如果您有自己的图标图片（PNG、JPG等）：

1. **准备图片**：
   - 推荐尺寸：256x256像素或更大
   - 格式：PNG（支持透明背景）
   - 设计：简洁明了，适合小尺寸显示

2. **转换为ICO格式**：
   ```python
   # 使用Python脚本转换
   from PIL import Image
   
   def convert_to_ico(input_path, output_path):
       img = Image.open(input_path)
       img = img.convert('RGBA')
       sizes = [(16, 16), (32, 32), (48, 48), (256, 256)]
       img.save(output_path, format='ICO', sizes=sizes)
   
   # 使用示例
   convert_to_ico('my_icon.png', 'src/assets/icon.ico')
   ```

3. **重新打包**：
   ```bash
   python scripts/build.py
   ```

### 🔧 故障排除

#### 图标不显示
1. 检查文件路径：`src/assets/icon.ico`
2. 验证ICO格式：使用图片查看器打开
3. 重新打包程序：`python scripts/build.py`

#### 图标模糊
1. 使用更高分辨率的原始图片
2. 确保ICO文件包含多种尺寸
3. 简化图标设计，避免过多细节

#### 打包失败
1. 检查图标文件是否损坏
2. 查看PyInstaller错误信息
3. 尝试删除图标重新打包
